# Todo App 
Todo App is a **full-stack web application** built using **React, Tailwind CSS, HTML5, Node.js, Express.js, and MongoDB**.This project allows users to efficiently manage their daily tasks with a clean interface and a powerful backend for data storage and processing.

# Features

- **Create Tasks** – Add new todos easily
- **View Tasks** – Display all tasks in a structured list
- **Delete Tasks** – Remove completed or unwanted tasks
- **Database Storage** – Tasks are stored in MongoDB for persistence
- **Real-time UI Updates** – Instant changes using React state management
- **Responsive Design** – Works across mobile, tablet, and desktop

# Technologies Used
# Frontend
- **React** – for building dynamic UI components
- **Tailwind CSS** – for modern and responsive styling
- **HTML5** – for structure and layout
- **Axios** – for API communication
# Backend
- **Node.js** – for server-side runtime
- **Express.js** – for API routing and handling requests
# Database
- **MongoDB** – for storing and managing todo data

# How It Works
1. User adds a new task from the frontend.
2. The request is sent to the backend API.
3. Backend processes the request and stores data in MongoDB.
4. Tasks are fetched and displayed dynamically on the UI.
5. Users can update or delete tasks, and changes are reflected instantly.
